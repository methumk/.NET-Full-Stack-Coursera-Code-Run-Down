Pseudo-Elements

Pseudo elements are used to style speficied parts of an element. E.g. inserting content before, after content of another element or styling first letter in an element.
This is different from nth-child in the sense that nth-child selects elements as a whole, where this targets a specific part in the element itself.
In addition, unlike pseudo-class selectors, pesudo elements are designated by double colons (::), but you can technically still use signle colon (until they introduce new pseudo elements).

::before

Format: <element>::before {}
You can add content before the main element.

Active

::after

Format: <element>::after {}
You can add content after the main element.

Active

::first-line and ::first-letter

Format <element>::first-line/letter {}
Like their name suggests they apply style to either the first line or the first letter

CSS Positions

Allows you to control the position of elements on a webpage and how they'll appear on to the user on a viewport.

Static

Static elements are default positions of all elements. They cannot be moved or repositioned with CSS.
This div block only has background color and height in CSS.
The above div is static because it's the default position. And you cannot move it with other move properties.

Relative

Relative positioning allows you to position an elemment relative to its normal position.
E.g. top, boottom,. left, right. These will let you choose how much space you want from the original top,bottom, left, right positions.
E.g. top: 20px will move the item 20px DOWN from the top of the static position. These will move AWAY from the positional property.

CSS Floats

Allows you to create multi column layouts and wrap text around images or other elements. Allows you to position an element on right or left side of its container.
This creates space for other content to flow around it, which can help you create multicolumn layouts or positoning images alongside text.
Often used in conjunction with the clear property which controls beahvior of elements following a floated element.
Float can have values left (elements float to the left of its container), right (float to right of container), none (element doesn't float displayed where it occurs in text) - default, inherit (inherits float of parent)

CSS Display

Used to specify the display behavior (type of rendering box used in an HTML element which affects how it's rendered) of an alement
  • Block: Creates a new line and takes up the full width of their container
  • Inline: Elements flow with text and take up as only as much width as necessary
  • Inline Block: Hybrid of the two, allow for text flow and can take additional space as needed.

CSS Units

Several different units for expressing a length. Length is a number followed length unit. Two major length units: absolute and relative.

Absolute length

These units are fixed and a length in that size will always be that size
e.g. cm, mm, px, etc

Relative Length

These units are relative to another length property. These scale better between different rendering mediums (depends on screen size) and should be preferred over absolutes
e.g. em, ex, %, etc
