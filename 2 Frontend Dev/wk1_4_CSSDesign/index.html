CSS Box Sizing

Allows you to control how the width and height of an element are calculated by including the padding and border into the element's total width and height calculations.
The default value of box-sizing is content value which means element's w/h calculated only based on content of element
This can cause layout issues if you add padding or border to the element as it increases size of the element.

If you use border box values, the padding and border get included. E.g. if element w is 100px, padding 10px and border 1px. Total width remains 100px.

Sizing conventions

  • grid-column/row-start:start space index; and grid-column/row-end: end space index;
    Specify the start and end space indexes that the item should span.
  • grid-column/row:start space idx / end space id;
    Shorthand to specify the start and end space indexes that the item should span.
  • grid-column/row:span #col/row;
    Specify the number of rows or cols that item should span

Object Fit and Position

Object fit and position are used to position an image/video inside an html element.
Object fit specifies how img/video tags should be resized to fit its container. This porperty tells content to fill the container in different ways e.g.l preserve aspect ration, etc.
Object position specifies how img/video tags should be positioned within its container.

object-fit: fill

Gray box is 200x200 px note how it takes up the entire space and the aspect ratio is messed up (letterboxed).

object-fit: contain

Gray box is still 200x200 note how it makes the image smaller to make the aspect ratio correct.

object-fit: cover

Gray box is still 200x200, cover sizes image to amintain aspect ratio and fills element's entire content box, if aspect ratio doesn't match then the image will be clipped to fit.

object-fit: scale-down

Gray box is still 200x200, scale-down sizes content as if none or contain were specified whichever is smaller is chosen. 
            
                #img{
                    width: 200px;
                    height: 200px;
                    object-fit: fill | cover | contain | scale-down | none (doesn't resize);
                }
